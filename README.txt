Remarques:
---------  

*Version python : Projet a été réalisé avec python 2.7. Aucune installation de bibliothèques externes n'est requise.

*les scripts :

GibbsFinalVersion.py : script contenant la classe GibbsSampler et ses méthodes.
IhmFinal.py : script contenant les détails de l'intérface graphique.







Programme :
-----------

*Script à exécuter : IhmFinal.py 
*L'intérface graphique :

Contient un bouton 'browse' pour récupérer le fichier à lire;
un bouton 'preview' pour afficher les sequences lu, ainsi que leur noms et nombre.
3 champs d'entrée (run : nombre de tours à faire; iteration: nombre d'itérations; W : longueur du motif ),;
une case à cocher pour definir l'alphabet des séquences (nucleic : pour 'ACTG' ; proteic : pour 'ARNDCQEGHILKMFPSTWYV' ).

Au lancement l'intérface graphique contient des valeurs par défaut que l'utilisateur pourra modifier.
